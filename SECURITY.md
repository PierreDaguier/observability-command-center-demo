# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability

Please do not open public issues for sensitive vulnerabilities.

1. Email: `security@observability-command-center-demo.local`
2. Include: impact, attack path, affected component, proof of concept.
3. Expected response:
   - Acknowledgement within 2 business days
   - Triage within 5 business days
   - Patch or mitigation timeline shared after triage

## Security Posture Notes
- Demo uses synthetic telemetry and non-production credentials.
- Default Grafana credentials must be changed outside local demo usage.
- No real customer data is stored in this project.
