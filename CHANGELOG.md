# Changelog

All notable changes to this demo are documented here.

## [1.0.0] - 2026-03-11
### Added
- Premium dark UI for KPI dashboard, correlation, alerts, and replay control.
- 7-minute client walkthrough and governance docs.
- Docker Compose stack with OTel, Prometheus, Grafana, Loki, and Tempo.

## [0.6.0] - 2026-03-11
### Added
- Alert center with severity timeline and incident grouping.
- Correlation flow linking traces, logs, metrics, and alerts.
- Incident replay scenarios (`checkout-lockstorm`, `auth-token-regression`).

## [0.1.0] - 2026-03-11
### Added
- Synthetic telemetry simulator (e-commerce + SaaS services).
- Backend aggregation API and seed dataset generation.
- Initial command center UI shell and service reliability cards.
